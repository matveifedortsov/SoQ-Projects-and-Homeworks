# SQExotic - Phoenix Autocallable Pricer

## Описание

Этот проект представляет собой инструмент для оценки стоимости экзотического опциона типа **Phoenix Autocallable** на две акции (Visa и Mastercard).

## Возможности

- **Монте-Карло симуляция**: Генерация коррелированных путей цен акций с использованием геометрического броуновского движения (GBM)
- **Расчёт греков**: Delta, Gamma, Vega через метод конечных разностей
- **Анализ рисков**: Стресс-тестирование и анализ чувствительности
- **Визуализация**: Графики распределения выплат, путей цен и результатов анализа

## Параметры продукта

- **Базовые активы**: Visa (V) и Mastercard (MA)
- **Срок**: 3 года
- **Купон**: 10% годовых
- **Автоколл**: 100% от начального уровня (ежеквартально)
- **Барьер защиты**: 90% от начального уровня (worst-of)
- **Наблюдения**: Ежеквартальные (12 наблюдений)

## Методология

### Генерация путей
Используется модель геометрического броуновского движения с корреляцией между активами:
- Холески декомпозиция для генерации коррелированных случайных величин
- Учёт дивидендных доходностей
- Дневные шаги симуляции

### Оценка выплат
1. Проверка условия автоколла на каждом наблюдении
2. Проверка барьера защиты (knock-in)
3. Расчёт финальной выплаты в зависимости от сценария

### Расчёт греков
- **Delta**: Изменение цены при изменении спота на ±1%
- **Gamma**: Вторая производная по споту
- **Vega**: Чувствительность к волатильности (±1%)

## Результаты
